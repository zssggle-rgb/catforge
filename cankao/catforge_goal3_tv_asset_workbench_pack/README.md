Purpose:
Turn the current pipeline-board MVP into a production asset workbench for TV category.

This goal does not replace Goal 1 core engine or Goal 2 production hardening.
It adds the missing product surfaces, data lineage, asset library management, SKU-level results, and calibration reports needed for real 1000-SKU delivery.

- TV asset libraries are visible, versioned, searchable, reviewable, and exportable as runtime assets.
- 1000 SKU analysis results are explorable and auditable.
- Calibration report proves coverage, price support, sales support, comment perception, and expert review quality.
